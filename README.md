##什么是Flight

Flight是一个快速，简易，可扩展的微型PHP框架。
Flight能够让你快速而简便的创建出RESTful的web应用。


```php
require 'flight/Flight.php';

Flight::route('/', function(){
    echo 'hello world!';
});

Flight::start();
```
